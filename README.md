# KNIME Data Preprocessing Workflow

## 📌 Project Overview
This project demonstrates a data preprocessing workflow built using KNIME Analytics Platform. It focuses on preparing raw data for analysis by applying essential data cleaning and transformation techniques.

## ⚙️ Workflow Steps
- Data import using CSV Reader  
- Missing value handling using statistical imputation (mean/median/mode)  
- Outlier detection using Box Plot visualization  
- Data aggregation using GroupBy  

## 🛠️ Tools Used
- KNIME Analytics Platform  

## 🎯 Objective
To build a structured data preprocessing pipeline that improves data quality and prepares datasets for further analysis or machine learning tasks.

## 📂 Project Structure
- workflow.knime → Main workflow file  
- dataset.csv → Input data (if included)

## 🚀 Key Concepts
- Data Cleaning  
- Outlier Detection  
- Data Transformation  
- Exploratory Data Analysis (EDA)
