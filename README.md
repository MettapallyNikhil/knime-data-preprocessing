# KNIME Data Preprocessing Workflow

## 📌 Project Overview
This project demonstrates a data preprocessing workflow built using KNIME Analytics Platform. It focuses on preparing raw data for analysis by applying essential data cleaning and transformation techniques.

## ⚙️ Workflow Steps
- Data import using CSV Reader  
- Missing value handling using statistical imputation (mean/median/mode)  
- Outlier detection using Box Plot visualization  
- Data aggregation using GroupBy  

## 🛠️ Tools Used
- KNIME Analytics Platform  

## 🎯 Objective
To build a structured data preprocessing pipeline that improves data quality and prepares datasets for further analysis or machine learning tasks.

## 📂 Project Structure
- treatment of Missing values.knwf → Main workflow file  
- categorical_dataset_with_missing_values.csv & production_dataset_with_missing_values.csv → Input data

## 🚀 Key Concepts
- Data Cleaning  
- Outlier Detection  
- Data Transformation  
- Exploratory Data Analysis (EDA)

## ▶️ How to Run

1. Open KNIME Analytics Platform  
2. Import the workflow file (.knwf)  
3. Execute the nodes step-by-step  
4. View results in output nodes  

## 📈 Results

- Missing values were successfully handled using statistical imputation
- No significant outliers detected in the dataset
- Data aggregation provided meaningful summary statistics
